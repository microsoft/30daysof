Welcome!​

The Connector Hackathon Welcome Message

Where to find us​

  • Blog - subscribe to the feed for announcements & updates
  • Roadmap - Our 30 Day schedule
  • Discussion - ask questions, interact with peers, show-and-tell!

Your First Actions​

  1. Gear up: You need to set up a Microsoft 365 Developer Account and Power Apps Developer Account for you to able to create your custom connector. Learn how to do this for free here
  2. Join the Challenge Today: Don't be left behind in the learning, this Cloud Skills Challenge contains modules that will give you skills to successfully work on a connector while you compete with your peers for awards.
  3. Check out the Microsoft Power Platform Connectors Repository Your projects at the end of the program will go in the 'independent-publisher-connector' folder.

Day 1: Summary​

As a code-first developer, where do you fit in the Low Code/No Code (LCNC) Environment (Power Platform)?

Fusion Teams - Roles & Responsibilities​

Fusion Teams Empower everyone to do more. A code first developer builds an API, which can then be made into a connector and in turn, connectors can be made into drag and drop components to be used by citizen developers in low code solutions

  1. Citizen Developers

    • Understand the business
    • Solve their own challenges using Technology

  2. IT Professionals

    • Enable high productivity app development.
    • Implement governance and compliance.

  3. Professional Developers

    • Reduce time to develop and deploy.
    • Use code to unblock complex requirements.

Participation Options​

  1. Identify an existing connector on GitHub and improve it!
  2. Use an already deployed API to build a connector! [Recommended]
  3. Build an API from scratch, deploy it and build a connector using it!
  • All independent publisher contributors are advised to go through the Independent publisher certification process guidelines before building & submitting their connectors
  • Participants who opt to improve existing connectors can ONLY improve connectors in the Independent-Publisher-Connectors folder in the repository

Incentives & awards​

  • All participants who submit a working connector will receive the M365 Platform Community - Independent Publisher 2022 Badge to add to their CV/portfolio

  • 5 Cloud Skills Challenge Random Winners will be awarded with a special SWAG Pack

  • The Top 4 Connectors will each receive the Hackathon Grand Prize
